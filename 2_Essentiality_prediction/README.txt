Here, the code to calculate predict essential genes from the calculated gMCSs and RNA-seq data.

RDSResults/ and RDSResults_adaptation/ folders do not contain the intermediate results of the DepMap analysis, that have been eliminated due to space storage limitations

These files can be obtained by executing these files in the indicated order:

 1. 01_generate_gene_information_gMCS_matrix_HART_new.R
 2. 02_prediction_of_essential_genes.R
 3. 03_adapation_calculation.m
 4. 04_adapation_to_essentiality.R
